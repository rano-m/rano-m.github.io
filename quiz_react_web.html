Dark-Theme Quiz React Native Web App

Code: GitHub

This program is a React Native application that creates a quiz screen with both multiple-choice and fill-in-the-blank questions.

The core of the application lies in the QuizScreen component, which orchestrates the state management, rendering logic, and user interactions. Questions are predefined within an array, each carrying essential information such as the question type, prompt, and answer options or correct response. Users engage with the quiz by selecting options for multiple-choice questions or typing in responses for fill-in-the-blank questions. Visual cues, such as background color changes, provide immediate feedback on the correctness of their answers.

This quiz is based on completion rather than testing knowledge for grading. The score state variable tallies correct answers, incrementing with each accurate response. A key feature is the useEffect hook, which monitors the score. When the score aligns with the total number of questions, the quiz is considered complete. This system not only informs users of their progress but also serves as a motivational element to strive for a perfect score.
