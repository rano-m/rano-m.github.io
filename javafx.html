"Find the Odd" Game

Code: GitHub

Find the Odd is a the project is crafted using Java, with a focus on leveraging the JavaFX framework for creating a dynamic and visually appealing user interface. The implementation follows Object Oriented Programming principles, emphasizing modularity and encapsulation. Through the utilization of JavaFX and OOP, the project provides an interactive and immersive gaming experience, challenging players to identify the odd image within a set timeframe.

The game begins with a Start Window, providing the player with essential information about the game and featuring a "Start Game" button. Once initiated, the interface transforms to reveal a dynamic layout comprising a Top HBox and a GridPane, both nested within a BorderPane. The HBox incorporates Time and Score Labels, each accompanied by descriptive text.

A crucial aspect of the project involves implementing a timer mechanism utilizing the TimeLine class, set to a default time of 60 seconds. Simultaneously, the player's score is initialized to zero. The game progresses as the player attempts to identify the odd image from a matrix of buttons, each embedded with dynamically generated images.

The images for the game are sourced from external files, containing pictures of animals with subtle differences. One image among the set is randomly designated as the odd image, contributing to the game's variability and challenge. The player's score increments with each correct identification, while incorrect selections do not affect the score.

Upon the timer reaching zero, the game interface is cleared, presenting a "Game Over" message, the player's final score, and an option to restart the game with a "Play Again" button. The project's codebase is hosted on GitHub, allowing for version control and collaborative development. The use of Java, JavaFX, and dynamic image loading contributes to an engaging and interactive gaming experience in the "Find the Odd" project.
